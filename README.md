Homy is a mobile application designed to help families and shared households organize shopping and plan meals together in a simple and efficient way.

## 📱 Overview
Homy allows multiple users to create and manage shared shopping lists in real time. It reduces forgotten items, duplicate purchases, and improves coordination between household members.

The app also supports basic meal planning by connecting meals with required grocery items.

## ✨ Features
- 📝 Shared shopping lists
- 👨‍👩‍👧‍👦 Multi-user collaboration
- 🔄 Real-time updates
- 🍽️ Meal planning integration
- 🌍 Multi-language support (Germany ,English & Arabic)

- ## 🚀 Tech Stack

- sqlite
- Firebase (optional: if used)
- Local Storage (SharedPreferences / SQLite / Hive)
